# Week 4 — NAFLD Liver Multi-omics

## Overview
Multi-omics analysis of Non-Alcoholic Fatty Liver Disease (NAFLD) using public datasets.

## Datasets
- **GSE89632** — Bulk RNA-seq, human liver biopsies (NAFLD vs healthy)

## Plan
1. Bulk RNA-seq: differential expression (NAFLD vs healthy)
2. Drug repositioning: identify candidate compounds
3. scRNA-seq: cell-type-specific expression
4. Spatial transcriptomics: tissue-level mapping
5. Network analysis: gene/pathway networks
